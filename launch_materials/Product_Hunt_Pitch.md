Hey Product Hunt! 👋 I'm Ishaan, the creator of Software MRI.

I built this tool to solve a massive pain point I've faced as an engineer: getting completely lost in massive codebases. When you join a new project with 5,000+ files, reading text top-to-bottom doesn't help you understand the *architecture*. 

So, I built a 3D visualization engine that turns your local codebase into a living, breathing digital organism. 

Software MRI parses your code locally (100% private) using ASTs, and renders a 3D galaxy using a custom WebWorker physics engine. It spots circular dependencies, highlights extreme complexity with red hazard signs, and traces execution flow.

**Key features:**
🚀 Blazing fast (renders 10,000 files at 60fps)
🔒 100% Local (no cloud, no telemetry)
🌐 Supports 300+ languages
✨ Spots dead code and structural flaws visually

I built this open-source because I believe we need better tools for codebase comprehension. I would love your feedback, and I'll be hanging out in the comments all day to answer technical questions!
